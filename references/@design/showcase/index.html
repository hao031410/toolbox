Design System

设计哲学

本设计系统遵循渐进引导式交互风格，由浅入深地引导用户完成任务，避免一次性暴露全部功能造成认知负担。

🎯
克制
颜色少而精，主色调不超过 3 个，保持界面简洁清晰
👁
清晰
信息层级通过颜色对比度明确区分，一眼识别重点
🔄
一致
相同语义元素在不同页面表现一致，降低学习成本
🪜
渐进
交互引导由浅入深，不一次性暴露全部功能

渐进引导设计原则
✓ 首屏极简 — 只展示核心入口
✓ 即时反馈 — 任何操作都有视觉响应
✓ 分步引导 — 复杂流程拆解为简单步骤
✓ 避免阻断 — 优先使用 inline 展开而非弹窗
