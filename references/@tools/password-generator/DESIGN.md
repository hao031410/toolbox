# 密码生成器设计文档

## 工具信息

- **名称**: 密码生成器
- **ID**: `password-generator`
- **类型**: 数据生成工具
- **进度**: 🚧 设计中

## 功能需求

### 核心功能
1. 生成随机密码
2. 支持自定义长度（8-64位）
3. 支持选择字符类型（大写、小写、数字、符号）
4. 一键复制到剪贴板
5. 密码强度指示

### 渐进引导流程

```
┌─────────────────────────────────────┐
│  🔐 密码生成器                       │
│                                     │
│  ┌─────────────────────────────┐   │
│  │      ••••••••••••••          │   │  ← 初始显示占位符
│  └─────────────────────────────┘   │
│                                     │
│  [    生成密码    ]                 │  ← 主要行动点
```

### 配置选项（展开后）

| 选项 | 类型 | 默认值 |
|------|------|--------|
| 密码长度 | 滑块 8-64 | 16 |
| 大写字母 | 开关 | ✅ |
| 小写字母 | 开关 | ✅ |
| 数字 | 开关 | ✅ |
| 特殊符号 | 开关 | ✅ |

## 交互设计

### 状态流转

```
初始状态 → 点击生成 → 显示密码 → 可复制
              ↓
         展开配置 ←→ 调整选项 → 重新生成
```

### 即时反馈

- 生成按钮：点击后显示加载动画 300ms
- 复制按钮：点击后显示"已复制"提示 2s
- 强度条：生成后动画显示

## 视觉规范

### 遵循 Design System

- 颜色：`--primary-900` 主按钮
- 输入框：`--bg-card` 背景，`--border-medium` 边框
- 滑块：自定义样式，使用 `--primary-700`
- 开关：使用标准 Toggle 组件
- 间距：按 `--space-*` 规范

### 组件清单

- `Button` - 生成按钮、复制按钮
- `Input` - 密码显示（只读）
- `Slider` - 长度选择
- `Toggle` - 字符类型开关
- `ProgressBar` - 强度指示
