说明

Cron 表达式用于指定定时任务的执行时间。默认按 6 段解析；填写年份时自动按 7 段解析。

┌────────── [可选] 秒 (0 - 59)
├────────── 分钟 (0 - 59)
├────────── 小时 (0 - 23)
├────────── 日 (1 - 31)
├────────── 月 (1 - 12)
├────────── 星期 (0 - 6 / SUN-SAT)
└────────── [可选] 年
  • 星号 `*`：表示匹配任意值。
  • 逗号 `,`：用于分隔多个值。
  • 斜线 `/`：用于指定间隔值。
  • 连字符 `-`：用于指定范围。
  • 问号 `?`：仅用于日期和星期字段，表示不指定具体值。

常见样例

  • 0 0 12 * * ?每天中午 12 点执行
  • 0 0/5 * * * ?每 5 分钟执行一次
  • 0 0 8-18 ? * MON-FRI周一到周五 8 点到 18 点之间每小时执行一次
  • 0 30 9 ? * MON,WED,FRI每周一、周三、周五 09:30 执行
  • 0 0 0 1 * ?每月 1 日 0 点执行
  • 0 15 10 L * ?每月最后一天 10:15 执行
  • 0 0 9 ? * 2#1每月第一个周一 9 点执行
  • 0 0 18 ? * SAT,SUN每周末 18 点执行
