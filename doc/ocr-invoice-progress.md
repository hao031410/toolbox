# OCR 发票工具进度

更新时间：2026-03-12

## 已完成

- 工具站 MVP 结构已稳定：
  - 首页
  - 计算器
  - 发票 OCR 汇总页
- OCR 上传链路已打通：
  - 支持 `zip/pdf/png/jpg/jpeg`
  - 支持 ZIP 递归解包
  - PDF 支持本地文本提取
  - 图片/扫描 PDF 已接入硅基流动 `deepseek-ai/DeepSeek-OCR`
- 发票结构化抽取第一版已完成：
  - 发票类型
  - 发票号
  - 开票日期
  - 购买方
  - 销售方
  - 不含税金额
  - 税额
  - 价税合计
  - 基础分类
- 导出能力已完成：
  - 前端支持 `xlsx`
  - 默认导出 3 个 sheet：
    - `票据明细`
    - `公司汇总`
    - `分类汇总`
  - 异常时回退 CSV
- 重复检测第一版已完成：
  - 支持明确重复检测：`发票号 + 销售方 + 日期 + 金额`
  - 支持疑似重复检测：`销售方 + 日期 + 金额`
  - 页面已支持复核态展示和导出

## 已修复

- 上传中文文件名乱码
- 电子发票买卖方优先按标签抽取
- PDF 本地解析与 OCR provider 双路径兼容
